# Benchmark report

## TL;DR
- Benchmark reports are generated by the internal benchmark scripts.
- Generated reports include `mae`, `rmse`, `rmae`, per-model `execution_time`, representative forecast plots, and optional PDF export.
- The committed docs page is only a placeholder.
- Regenerate benchmark outputs locally when you need fresh numbers.

## Regenerate a report

```bash
uv run --frozen python scripts/benchmark_long_horizon.py run \
  --mode daily \
  --output logs/benchmark_long_horizon_daily.csv \
  --output-md docs/benchmark.md \
  --output-pdf docs/benchmark.pdf
```

The benchmark compares the default-parameter versions of:
- `TimeBase`
- `TimeBaseTrend`
- `DLinear`
- `NLinear`
- `SeasonalNaive`
- `MFLES`
